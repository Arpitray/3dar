🎯 Generate AR Marker (.mind file)

⚠️ Required for AR Mode: Your app needs a .mind compiled marker file to detect images in AR mode.

📋 Quick Steps:

Step 1: Go to MindAR Compiler
🔗 https://hiukim.github.io/mind-ar-js/tools/compile
Step 2: Upload marker image
Upload a clear image (JPG/PNG) you want to use as a marker:
  • QR code (easiest to test)
  • Business card / ID card
  • Product packaging
  • Book cover
  • Any distinct image with features
Recommendation: Use a business card or product photo for best results.
Step 3: Download the .mind file
The compiler will generate a .mind file automatically.
Step 4: Upload to your server
Move the downloaded .mind file to: public/targets/card.mind

Via File Explorer:
  1. Find your downloaded .mind file
  2. Copy it
  3. Navigate to your project: public/targets/
  4. Paste and rename to: card.mind
Step 5: Restart your dev server
npm run dev --host
Then reload your app and switch to AR mode!
🧪 Testing Tip: Start with a simple QR code or business card. More complex images take longer to train but give better tracking.

✅ Once Complete:
  1. Refresh your app: go back to Demo mode first
  2. Click "Switch to AR"
  3. Grant camera permissions
  4. Point camera at your marker image!
